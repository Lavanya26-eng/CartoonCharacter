# 🎭 Cartoon Character Quiz

A fun and interactive **Cartoon Character Quiz** built using **HTML, CSS, and JavaScript**. The application presents multiple-choice questions about popular cartoon characters, evaluates the user's answers, calculates the score, and displays a personalized result at the end.

## ✨ Features

- Interactive multiple-choice quiz
- Questions about popular cartoon characters
- Instant score calculation
- Displays a final performance message
- Attractive and colorful user interface
- Responsive and beginner-friendly design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
CartoonCharacterQuiz/
│── cartooncharacter.html
└── README.md
```

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **cartooncharacter.html** or open it in any web browser.
4. Answer each question by selecting an option.
5. Click the **Next** button to proceed.
6. View your final score and performance after completing the quiz.

## 📖 Quiz Topics

The quiz includes questions about famous cartoon characters such as:

- SpongeBob SquarePants
- Mickey Mouse
- Doraemon
- Tom and Jerry
- Pluto
